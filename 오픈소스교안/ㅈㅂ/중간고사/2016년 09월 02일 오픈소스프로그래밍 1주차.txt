git 사용법
git bash 를 사용
git init 		로 초기화를 한다
git status 		로 상태 파악
git add 파일이름	으로 스테이지에 올림
git commit -m "메세지"	로 커밋함
git log 		로 버전관리를 볼수있음 (커밋의 횟수만큼)
git diff		워킹트리에 있는 내용과 커밋에 제출된 내용이 같은지 다른지 확인할수 있다
